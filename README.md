# Castorea-2
Project for the 2025-2027 years of competition leading up to ISR 19.

This is a preliminary test for another file management system alternative to the shared drive system that OSU HPS is currently using. Per concerns raised by members regarding multiple people uploading files at the same time to the system with the potential for one person to upload their updated file when working on one part and uploading their whole local NX file system before another person who worked on and updated a different part overrides their upload by updating their local NX file system to the one on the drive, we figured that this overall would be safer. Push protections, access restrictions, among other things will be needed before this can be utilized on a club wide level, aside from having all club members make GitHub accounts and be added to the Github with officers managing ownership of the GitHub organization. Nevertheless, this will be a good tool for project managers and eventually team leads to manage their sections of the project and automatically track revisions on individual parts without needing to keep a running archive folder. 

The hope is that when the project for ISR 20 comes around that this will be a more widely used tool. For now, it will be used as an optional tool to be ran as a backup to the NX file system on the drive with regular uploads from the file system on the drive onto the Github.
